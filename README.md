9. **目标代码生成**：介绍目标代码生成的过程和方法。

- **设计实例**：书中包含两个小型编译程序的设计实例，分别是PL/0语言编译程序和简单面向对象语言Decaf的编译程序。这些实例可作为课程设计的素材。
